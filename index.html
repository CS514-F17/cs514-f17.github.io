Introduction to Computer Science II
Spring 2017


Course Description

Design and development of significantly sized software using top-down design and bottom-up implementation. Dynamically allocated data, object-oriented programming, architecture of memory, basics of language translation, and basics of algorithm analysis. Development of simple graphical user interfaces. Four hours lecture.

Prerequisite

CS 110 with a minimum grade of C
